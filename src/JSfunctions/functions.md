## Functions

These are re-usable piece of code that perform specific tasks.

They are essential for:

1.Code reusability: Write once and use many times.

2.Code organization: Break complex problems into smaller parts.

3.Maintainability: Easier to debug and maintain.

4.Modularity: Build complex applications from simple pieces.

## Concepts:

Function declaration
Function expression
Arrow functions
Scope
Hoisting
